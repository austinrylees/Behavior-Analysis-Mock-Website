What is a behavior?

A behavior is anything that we do! Jumping, Dancing, Singing, Swimming, Flying,Typing. Anything and everything we do! An easy way to see if it is a behavior is to ask if a dead man could do it. For example, a dead man could "get " hit (be a recipient of external forces), lay, do nothing, and alternatively, a dead man cannot dance, sing, sit upright, etc. Another helpful tidbit would be to ask if we filmed you,could we observe it?

What is a Function of behavior?
A function explains the relationship between the environment and the behavior. In a universe in which we can assume that there is casuality or that there are reasons for things and that behaviors result from antecedents (fancy word for what happended right before a behavior) and a conesquences If it is safe to assume cause-and -effect relationships exist then the function would be the environmental explanation for a behavior
